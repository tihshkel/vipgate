# Папка для иконок

Эта папка предназначена для хранения иконок, которые используются в проекте.

## Форматы иконок

Рекомендуемые форматы:
- SVG (предпочтительно) - векторные иконки, масштабируются без потери качества
- PNG - растровые иконки с прозрачностью
- ICO - для favicon

## Использование

Иконки из этой папки можно импортировать в компонентах:

```jsx
import iconName from '../assets/icons/icon-name.svg'
```

Или использовать через компонент Icons.jsx, который находится в `src/components/Icons.jsx`.
