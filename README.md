# Vipgate.com - VIP-сервисы в аэропортах

Точная копия дизайна сайта Vipgate.com, созданная с использованием React и Tailwind CSS.

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск проекта

Для запуска проекта в режиме разработки:
```bash
npm run dev
```

Откройте браузер и перейдите по адресу, который будет показан в терминале (обычно http://localhost:5173)

## Сборка проекта

Для создания production сборки:
```bash
npm run build
```

## Структура проекта
